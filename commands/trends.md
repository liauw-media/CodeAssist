### Analysis Frameworks

#### Technology Adoption Curve
```
Innovators (2.5%) → Early Adopters (13.5%) → Early Majority (34%) → Late Majority (34%) → Laggards (16%)

Where is [technology/trend]?
- [ ] Innovator phase (experimental)
- [ ] Early adopter phase (growing)
- [ ] Early majority (mainstream)
- [ ] Late majority (commoditized)
- [ ] Laggard phase (declining)
```

#### Trend Lifecycle
```
Emergence → Growth → Peak → Plateau → Decline

Stage indicators:
- Emergence: <5 major players, niche coverage
- Growth: VC funding, growing coverage, new entrants
- Peak: Mainstream media, market saturation signals
- Plateau: Consolidation, established players
- Decline: Reduced investment, technology replacement
```

## Research Protocol

### Step 1: Define Scope
- What question are we answering?
- What decisions will this inform?
- What's the time horizon?

### Step 2: Gather Data
```
Primary research:
- Web search for recent articles
- Check industry reports
- Review competitor activity

Secondary research:
- Academic papers
- Government data
- Trade publications
```

### Step 3: Analyze Patterns
- What's growing vs declining?
- Who's investing?
- What's the adoption rate?
- What are leading indicators?

### Step 4: Synthesize Insights
- Key findings (3-5 max)
- Confidence levels
- Implications for decision
